What is Ethereum?

以太坊是一個分散的平台，可以運行智能合約，這些應用程序可以完全按照編程的方式運行，而不會造成停機，審查，欺詐或第三方乾擾的可能性。

See our website or read the docs for more infos!

什麼是以太坊（Go Ethereum）？

以太坊Go是以太坊協議的三個原始實現之一（以及C ++和Python）。 它使用Go編寫，完全開源，並在GNU LGPL v3下獲得許可。

See our repository and downloads section for the code!

我如何使用它？

Go Ethereum既可以作為名為Geth的獨立客戶端使用，也可以安裝在幾乎任何操作系統上，也可以作為可以嵌入Go，Android或iOS項目中的庫使用。

See our installation guide for details!
